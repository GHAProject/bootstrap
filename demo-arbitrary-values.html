Bootstrap Arbitrary Values Demo

This page demonstrates the new arbitrary value functionality for Bootstrap spacing utilities.

Margin Examples

Pixel Values

mt-[300px] - 300px top margin

Percentage Values

mt-[25%] - 25% top margin

Rem Values

mt-[2rem] - 2rem top margin

Viewport Units

mt-[10vh] - 10vh top margin

Padding Examples

p-[300px] - 300px padding all sides
p-[25%] - 25% padding all sides
px-[2rem] - 2rem horizontal padding
py-[10vh] - 10vh vertical padding

Directional Examples

ms-[300px] - 300px margin start (left)
me-[25%] - 25% margin end (right)
mx-[2rem] - 2rem horizontal margins
my-[10vh] - 10vh vertical margins

Negative Margins

Note: Negative margins can cause overlapping. Use with caution.

-mt-[300px] - Negative 300px top margin
-ms-[2rem] - Negative 2rem start margin

Comparison with Standard Bootstrap

Standard Bootstrap

mt-5 - Bootstrap's largest margin (3rem)
p-5 - Bootstrap's largest padding (3rem)

Arbitrary Values

mt-[300px] - Custom 300px margin
p-[300px] - Custom 300px padding

Use Cases

Hero Section Spacing

py-[10vh] px-[25%]
Perfect for hero sections with viewport-based spacing

Precise Layout Control

mt-[300px] mb-[2rem]
Mix arbitrary and standard values for precise control

Responsive Design Elements

mx-[auto] p-[25%]
Centering with percentage-based internal spacing

Browser DevTools Inspection
